# Play Framework Example

A simple **Play Framework (Scala)** application demonstrating:

- Form handling
- External API integration (currency exchange)
- Basic persistence using a relational database
- MVC structure (Controller + Repository)

---

## 🚀 Overview

This project allows users to:

1. Enter an amount in USD
2. Choose a target currency
3. Convert it using a real-time exchange rate API
4. Store and display past conversions
